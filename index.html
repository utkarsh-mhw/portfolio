WhyHere

Applied ML

Smart neighborhood recommendation system for Atlanta that ranks areas based on user priorities like safety, transit, rent, and amenities. Uses collaborative filtering to suggest neighborhoods liked by similar users and a distance-decay scoring model enhanced with custom feature weighting to provide explainable rankings. Integrated data from 12+ APIs and processed it using Databricks and PySpark. Visualized results with an interactive map.
